2. Hreflang Tags

Added hreflang tags to tell Google that Arabic and English pages are translations of each other, preventing duplicate content issues.

Tags Added

  • <link rel="alternate" hreflang="ar"> — Arabic version
  • <link rel="alternate" hreflang="en"> — English version
  • <link rel="alternate" hreflang="x-default"> — Default (Arabic)
  • <meta property="og:locale"> — ar_SA / en_US
  • <meta property="og:locale:alternate"> — Alternate locale

File Modified

  • resources/views/front/partials/_head.blade.php
